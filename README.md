# 🚀 Nathaniel Tariman | Portfolio Website

A clean, modern, responsive portfolio website built from scratch using semantic HTML5, custom CSS3 variables/grids, and vanilla JavaScript. This project showcases my education, technical skills, certifications, and developer projects with a seamless light/dark mode experience.



---

## 📸 Preview & Design System

*   **Theme:** Modern Tech Blue & Slate White
*   **Typography:** Roboto (300, 400, 500, 700) via Google Fonts
*   **Layout Structure:** Adaptive CSS Grid & Flexbox
*   **Interactions:** Dynamic light/dark mode theme persistence via `localStorage`

---

## 🛠️ Tech Stack & Features

*   **Frontend Architecture:** HTML5 Semantic Markup, CSS3 Layouts (Grid/Flexbox)
*   **Interactivity:** JavaScript (ES6+) DOM manipulation
*   **Key Features:**
    *   🔄 **Stateful Dark Mode:** Remembers your viewing preference across refreshes.
    *   💼 **LinkedIn-Style Timeline:** Elegantly structured education history tracking.
    *   📱 **Fully Responsive:** Fluid layouts designed to work effortlessly on mobile devices, tablets, and wide desktop displays.
